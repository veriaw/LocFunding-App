# LocFunding

A mobile crowdfunding application built to connect local communities with potential donors through Location-Based Services (LBS). While working on this project, I noticed that many social initiatives struggle to gain visibility and funding, while potential donors often have limited awareness of communities in need around them.

To bridge this gap, I developed a platform that allows users to discover and support nearby social projects based on their geographic location. By combining crowdfunding with location awareness, the application encourages stronger engagement between donors and local communities.

![LocFunding](.documentation/uiux-crowdfunding.png)

## Features

* 🔐 Secure Authentication
* 💰 Funding Campaign Creation
* ❤️ Donation Management
* 🔔 Real-Time Notifications
* 📍 Location-Based Funding Discovery
* 🗺️ Interactive Map Integration
* 💱 Currency Conversion
* 🌍 Time Zone Adjustment
* 📱 Shake-to-Random Funding Recommendation
* 👤 User Profile & Donation History

## Tech Stack

* Frontend: Flutter (Dart)
* Backend: RESTful API
* Local Storage: Hive, Shared Preferences
* Security: RSA Encryption
* IDE: Visual Studio Code

## Goals

* Improve accessibility to social funding opportunities
* Help local communities gain visibility and support
* Connect donors with nearby social initiatives
* Encourage community-driven impact through technology
* Provide a secure and user-friendly donation experience
